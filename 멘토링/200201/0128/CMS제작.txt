무엇을? 개인 포트폴리오(git기반) 웹페이지를 생성해주는 웹페이지 제작
1. 사이트 가입
2. 테마선택
3. git을 알려주면 pull해서 데이터 가져옴
4. 프로젝트/공부한내용/내정보 <-사용자가 입력할 부분
git repository 에서 공부내용인지 프로젝트인지 나누는건 사용자가 하고
웹페이지에는 사용자정보, 프로젝트들, 공부내용 예쁘게 정리(테마에 맞춰서)
5. 기본적으로 만들어주고 수정하고싶은부분이 있을 수 있기때문에 사용자가 조금 수정하면 끝

어떻게?
- AWS로 웹서버 띄움(부모웹페이지)
- 호스팅할당..이것도 AWS에 인스턴스올려서
- 웹페이지는 HTML,CSS이용해서 디자인, Javascript,jQuery로 웹사이트 다이나믹하게, FTP 클라이언트를 사용하여 로컬파일을 서버에 업로드(소스코드 서버에 업로드)

왜?
워드프레스나 줌라, Drupal 같은 CMS(웹 프레임워크)를 만들고싶다.
이런 사이트들은 예쁜 테마를 제공하고 데이터들은 사용자가 직접 입력해야하는데 나는 git으로 데이터를 받은 상황에서 사용자가 더 수정할 필요 없이
바로 웹사이트를 만들고 싶다.
